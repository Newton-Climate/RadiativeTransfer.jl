RadiativeTransfer Module

Introduction

The RadiativeTransfer module allows end-to-end simulation of radiative transfer (RT) throughout Earth's atmosphere and surface. Specifically, it:

  1. Enables 1D vectorized plane-parallel RT modeling based on the Matrix Operator Method.
  2. Incorporates fast, high fidelity simulations of scattering atmospheres containing haze and clouds – including pressure- and temperature-resolved absorption profiles of gaseous species in the atmosphere.
  3. Enables GPU-accelerated computations of the resulting hyperspectral reflectances/transmittances.

Example

using RadiativeTransfer
using RadiativeTransfer.vSmartMOM

## 
## STEP 1: Load / Customize RT parameters
## 

# If you would like to load your own parameters from a YAML file
# (See required format at: https://github.com/RupeshJey/RadiativeTransfer.jl/blob/master/src/vSmartMOM/ModelParameters/DefaultParameters.yaml)
parameters = parameters_from_yaml("RadiativeTransfer/src/vSmartMOM/ModelParameters/DefaultParameters.yaml")

# OR if you would like to load a default set of parameters
parameters = default_parameters();

# You can then change any individual fields in parameters (parameters.field = ...)

## 
## STEP 2: Create a model from parameters
## 

# Creating this model will use the parameters object to calculate derived fields and attributes
model = model_from_parameters(parameters);

# Again, you can then change any derived field in model (model.field = ...)

## 
## STEP 3: Run the RT simulation
## 

R = rt_run(model)

struct vSmartMOM_Parameters

A struct which holds all initial model parameters (before any computations)

Fields

  • float_type

    Float type to use in the RT (Float64/Float32)

  • λ_band

    band center wavelength (nBand)

  • λ_ref

    reference wavelength

  • depol

    Depolarization value

  • l_trunc

    Truncation length for legendre terms (scalar for now, can do nBand later)

  • Δ_angle

    Exclusion angle for forward peak

  • max_m

    Hard cutoff for maximum number of m iterations

  • polarization_type

    Type of polarization (I/IQ/IQU/IQUV)

  • obs_alt

    Altitude of observer [Pa]

  • sza

    Solar zenith angle [deg]

  • vza

    Viewing zenith angles [deg]

  • vaz

    Viewing azimuthal angles [deg]

  • nAer

    Number of aerosol species

  • τAer_ref

    AOD at Reference wavelength (nAer)

  • μ

    Log mean radius (nAer)

  • σ

    Log stddev of radius (nAer)

  • r_max

    Maximum radius

  • nquad_radius

    Number of quadrature points for integration of size distribution

  • nᵣ

    Real part of refractive index (nBand,nAer)

  • nᵢ

    Imag part of refractive index (nBand,nAer)

  • p₀

    Pressure peak Pa

  • σp

    Pressure peak width Pa

  • file

    Path to atmospheric profile file

  • timeIndex

    Time index to retrieve from file

  • lat

    Latitude of atmospheric profile

  • lon

    Longitude of atmospheric profile

  • profile_reduction_n

    Length of profile reduction

  • spec_grid_start

    Starting wavelength for absorption grid (nBand)

  • spec_grid_end

    Ending wavelength for absorption grid (nBand)

  • spec_grid_n

    Number of spectral points in absorption grid (nBand)

  • molecules

    Molecules to use for absorption calculations (nBand, nMolecules)

  • broadening_function

    Type of broadening function (Doppler/Lorentz/Voigt)

  • wing_cutoff

    Wing cutoff to use in cross-section calculation

  • CEF

    Complex Error Function to use in Voigt calculations

  • brdf

    Surface reflectance type

  • decomp_type

    Algorithm to use for fourier decomposition (NAI2/PCW)

  • quadrature_type

    Quadrature type for RT streams (RadauQuad/GaussQuadHemisphere/GaussQuadFullSphere)

  • architecture

    Architecture to use for calculations (CPU/GPU)
